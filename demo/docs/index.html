Meridian Web SDK Documentation

Overview

You will need to obtain a read-only API token from the Meridian Editor before you can use the Meridian Web SDK. 

    
      MeridianSDK.init({
        api: MeridianSDK.createAPI({
          environment: "production",
          token: "{{ INSERT_YOUR_READONLY_TOKEN_HERE }}"
        })
      });

      var element = document.getElementById("meridian-map");
      var map = MeridianSDK.createMap(element, {
        locationID: "{{ INSERT_YOUR_LOCATION_ID_HERE }}",
        floorID: "{{ INSERT_YOUR_FLOOR_ID_HERE }}"
      });

      map.update({
        // ...
      });

You can obtain the location ID and floor ID for a given map by navigating to it in the editor and grabbing the IDs from the URL in your browser.
